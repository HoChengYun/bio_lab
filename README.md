# ECG 信号处理实验 - 使用指南

## 📋 项目概述

本项目实现了基于 STM32F401 和 MAX86150 传感器的实时 ECG 信号采集、处理和离线心率计算系统。

**主要功能**:
- ✅ 实时 ECG 信号采集 (400 Hz 采样率)
- ✅ 带通滤波 (0.5-50 Hz)
- ✅ 离线 R 波峰值检测
- ✅ 心率计算 (30-200 bpm 范围)
- ✅ 数据导出 (CSV / NPY 格式)
- ✅ 频谱分析 (FFT)

## 🔍 信号处理详解

### 1. 带通滤波器

**设计参数**:
- 下截止频率: 0.5 Hz (去除基线漂移)
- 上截止频率: 50 Hz (去除高频噪声和工频干扰)
- 类型: Butterworth 4阶
- 方法: 双向滤波 (filtfilt - 零相位失真)

**效果**:
```
原始信号 (含噪声、漂移)
        ↓
   带通滤波
        ↓
    干净信号 (R波清晰)
```

### 2. 信号标准化

```
标准化后的信号 = (原始 - 平均值) / (标准差 + ε)
```

目的: 使峰值检测阈值适用于不同幅度的信号

### 3. R 波峰值检测

使用 scipy.signal.find_peaks():

```python
峰值检测条件:
- 相对高度 > 0.5 (标准化后)
- 相邻峰值最小间隔 >= 133 点 (~0.33秒)
  对应最大心率约 180 bpm
```

### 4. 心率计算

```
心率 (bpm) = 60 / 平均R-R间隔(秒)

例如: 若R-R间隔 = 0.833秒
      心率 = 60 / 0.833 = 72 bpm
```

## 📊 输出文件说明

### 1. 可视化报告 (PNG)

包含 6 个子图:
1. **原始信号** - 显示基线漂移和噪声
2. **滤波信号** - 带通滤波后的干净信号
3. **R 波检测** - 标记出检测到的 R 波峰值
4. **频谱图** - FFT 分析，显示频率成分
5. **标准化信号** - 显示峰值检测阈值
6. **统计信息** - 汇总数据和计算结果

### 2. 文本总结 (TXT)

包含:
- 信号基本信息 (采样率、时长、点数)
- 原始信号统计 (最大值、最小值、均值等)
- R 波检测结果 (数量、R-R间隔)
- 计算的心率值
- 频谱信息

### 3. 数据文件 (CSV/NPY)

**CSV**: 易于在 Excel 中查看和编辑
**NPY**: NumPy 二进制格式，便于Python进一步处理

```python
# Python 中加载和使用 NPY 数据
import numpy as np
data = np.load('ecg_data.npy')
```

## 🎯 参数调整指南

若需调整程序性能，修改以下参数:

### 1. 采样率 (STM32_UART.py)
```python
SAMPLE_RATE = 400  # Hz，不建议修改（固件相关）
```

### 2. 缓冲区大小
```python
BUFFER_SIZE = SAMPLE_RATE * WINDOW_SECONDS  # 显示时长
WINDOW_SECONDS = 5  # 显示 5 秒数据
```

### 3. 滤波参数 (ECG_Analyzer.py)
```python
ECG_LOWCUT = 0.5    # Hz，降低此值可去除更低频的漂移
ECG_HIGHCUT = 50    # Hz，提高此值可保留高频信息（可能增加噪声）
FILTER_ORDER = 4    # 滤波器阶数，提高可增强滤波，但增加延迟
```

### 4. 峰值检测参数
```python
height=0.5                      # 标准化后的高度阈值
distance=SAMPLE_RATE//3         # 相邻峰值最小间隔（点数）
```

**调整建议**:
- 若漏检 R 波 → 降低 `height`
- 若误检高频噪声 → 提高 `height`
- 若出现双峰 → 增加 `distance`

### 5. 心率计算窗口
```python
HR_WINDOW_SIZE = SAMPLE_RATE * 10  # 每 10 秒计算一次心率
# 改为 SAMPLE_RATE * 5 则每 5 秒计算一次
```

## 🐛 常见问题解决

### Q1: 程序启动报错 "No module named 'PyQt5'"
**A**: 安装依赖
```bash
pip install PyQt5 pyqtgraph scipy numpy pandas matplotlib
```

### Q2: 波形显示抖动或不稳定
**A**: 检查以下几点
- 确保 USB 连接稳定
- 降低采样率或增加滤波器阶数
- 检查电源，确保供电充足

### Q3: 计算的心率值异常 (如 > 200 bpm)
**A**: 可能原因
- 信号质量差（噪声多，基线漂移）
- 峰值检测阈值设置不当
- 设备接触不良

**解决方案**:
1. 检查传感器贴装位置和接触质量
2. 调整滤波参数
3. 调整峰值检测高度参数

### Q4: 无法识别串口
**A**: 
- 检查 USB 线连接
- 安装 CH340/PL2303 驱动（如需要）
- 在设备管理器中查看 COM 口号

### Q5: 导出文件打不开
**A**: 
- CSV 文件用 Excel 或记事本打开
- NPY 文件需要用 Python/NumPy 打开
```python
import numpy as np
data = np.load('file.npy')
print(data)
```
