## 二、实验原理

### 2.1 ECG 信号特性

- **采样率**: 400 Hz
- **主要频率成分**: 0.5-50 Hz
- **R 波特征**: 
  - 振幅最大（正向）
  - 代表心脏心室除极
  - 相邻 R 波间隔对应一个心跳周期

### 2.2 信号处理流程

```
原始 ECG 信号 → 带通滤波 → 信号标准化 → 峰值检测 → 心率计算
```

### 2.3 核心算法详解

#### 带通滤波器 (Butterworth Filter)
- **设计参数**:
  - 下截止频率: 0.5 Hz (滤除低频漂移)
  - 上截止频率: 50 Hz (滤除高频噪声和工频干扰)
  - 滤波器阶数: 4 阶
  
- **传递函数**:
$$H(z) = \frac{a_0 + a_1z^{-1} + a_2z^{-2}}{1 + b_1z^{-1} + b_2z^{-2}}$$

#### 信号标准化
$$x_{norm} = \frac{x - \mu}{\sigma + \epsilon}$$

其中: $\mu$ = 均值, $\sigma$ = 标准差, $\epsilon$ = 避免除零

#### R 波峰值检测
- 使用 `scipy.signal.find_peaks()` 函数
- **检测条件**:
  - 相对高度阈值: 0.5 (标准化后)
  - 相邻峰值最小间隔: $\frac{400}{3} \approx 133$ 点 (对应 ~130 bpm 最大心率)

#### 心率计算
$$HR = \frac{60}{\Delta t}$$

其中 $\Delta t$ 为相邻 R 波之间的平均时间间隔（秒）

2. 通过 USB 串口连接电脑与开发板

1. **编译烧录**: 使用 STM32CubeIDE 编译并烧录固件
2. **数据采集**: Python GUI 程序实时接收 ECG 数据
3. **实时处理**: 
   - 每接收 3000 个采样点 (~7.5 秒) 进行一次心率计算
   - 使用带通滤波器去除噪声
   - 使用峰值检测识别 R 波
   - 根据 R 波间隔计算瞬时心率

### 4.3 数据采集参数
- 采样率: 400 Hz
- 单次心率计算窗口: 7.5 秒 (3000 点)
- 滤波器带宽: 0.5-50 Hz
- 数据文件格式: CSV 和 NPY (NumPy)

### A. 代码片段 (关键函数)

```python
def _process_ecg(self, value):
    """訊號處理與心率計算"""
    self.ecg_buffer.append(value)
    self.hr_update_count += 1
    
    # 每3000點計算一次心率 (7.5秒)
    if self.hr_update_count >= 3000 and len(self.ecg_buffer) > 2000:
        self.hr_update_count = 0
        
        # 濾波
        signal = np.array(list(self.ecg_buffer))
        filtered = filtfilt(self.b, self.a, signal)
        
        # 標準化
        norm_signal = (filtered - np.mean(filtered)) / (np.std(filtered) + 1e-6)
        
        # 峰值檢測 (R波)
        peaks, _ = find_peaks(norm_signal, height=0.5, distance=SAMPLE_RATE//3)
        
        # 計算心率
        if len(peaks) >= 2:
            intervals = np.diff(peaks) / SAMPLE_RATE
            hr = 60.0 / np.mean(intervals)
            if 30 < hr < 200:  # 合理範圍
                self.hr_calculated.emit(float(hr))
```
